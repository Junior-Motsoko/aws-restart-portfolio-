# Labs Overview

This section contains hands-on AWS labs organized by service category.

## Categories

### 🖥️ Compute
Labs focused on EC2, Auto Scaling, and compute services.

### 📊 Databases
Database management labs including RDS and DynamoDB.

### 🐧 Linux
Linux system administration and shell scripting labs.

### 🌐 Networking
VPC, networking, and connectivity labs.

### 🔒 Security
Security best practices, IAM, and encryption labs.

### 💾 Storage
S3, EBS, EFS, and storage management labs.
